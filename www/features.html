Discworld mudlib version wibble features

Discworld release wibble:

  • extremely flexible and controllable command parsing system.
  • object matching parser for matching multiple objects, fully integrated into the mudlib.
  • complex and wonderful room system.
  • terrain system for generating areas based on template rooms.
  • several different sorts of shops.
  • exciting combat system.
  • family system for family names.
  • complex club system.
  • rcs version control integrated into the mudlib.
  • nice error reporting and management tools.
  • more wombles and bings per line of code.
  • mudlib from a successful running mud, in development since 1991.
  • fully featured player command set.
  • detailed and complete creator command set.
  • uses an automated documentation generation system to produce documentation for all the mudlib code.
  • lots of documentation on creator commands/player commands and code.
  • more poems/line of code than any other mudlib.
  • flexible race and guild system.
  • easily expandable set of effects to modify players behaviour and appearance.
