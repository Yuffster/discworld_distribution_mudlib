Discworld's distribution mudlib

This is the new release wibble of the Discworld mudlib. An extremely featureful mudlib from one of the old muds.

To run this server you will need the following:

  • Discworld v22.1b23 driver.
  • Installation instructions.
Options pieces to make your life easier:
  • Mysql for the errors system. collaboration tool.
  • The Discworld RCS system (to add source control).
  • Twiki a nice online development
Other features of this web server can be seen in the secure section of the pages.
